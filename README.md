# Go Gin Example

A simple REST API built with Go and Gin framework.

## Requirements

- Go 1.24.5 or higher

## Quick Start
1. **Install dependencies:**
   ```bash
   go mod tidy
   ```

2. **Run database migrations:**
   ```bash
   make db-migrate
   ```

3. **Start the server:**
   ```bash
   make dev
   ```

The API server will be available at `http://localhost:8080`

## Available Commands

```sh
make help
```
